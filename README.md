# grunt-css-splitter

> This is used for extracting parts of css to another files, kind of reverse engineering of concatening files. This is usefull in case you use SASS placeholders, and want to have it in other file, like typo, colors and whatever...

## Getting Started
This plugin requires Grunt `~0.4.2`

If you haven't used [Grunt](http://gruntjs.com/) before, be sure to check out the [Getting Started](http://gruntjs.com/getting-started) guide, as it explains how to create a [Gruntfile](http://gruntjs.com/sample-gruntfile) as well as install and use Grunt plugins. Once you're familiar with that process, you may install this plugin with this command:

```shell
npm install grunt-css-splitter --save-dev
```

Once the plugin has been installed, it may be enabled inside your Gruntfile with this line of JavaScript:

```js
grunt.loadNpmTasks('grunt-css-splitter');
```

## The "css_splitter" task

### Overview
In your project's Gruntfile, add a section named `css_splitter` to the data object passed into `grunt.initConfig()`.

```js
grunt.initConfig({
  css_splitter: {
    your_target: {
      // Target-specific file lists and/or options go here.
    },
  },
});
```
